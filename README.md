# Kepler Classifier

### System of exoplanets discovered by Kepler telescope classification.

*Made for masters thesis.*

##### Used classifiers:

- Decision Tree
- Random Forest
- Support Vector Machine
- K-nearest Neighbors

##### Technologies:

- Python, Pandas, Sklearn, Flask
- Vue 3 with Composition API, Axios, Plotly.js, SCSS
- .NET 6
